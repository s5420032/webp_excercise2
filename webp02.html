日本大学文理学部情報科学科 Webプログラミングの演習課題

the strong element

説明

内容の重要性、重大性、または緊急性が高いテキストを表します。ブラウザーは一般的に太字で描画します。

マークアップ例

This is<strong>not</strong>a pen.

実行結果

This is not a pen

the mark element

説明

周囲の文脈の中でマークを付けた部分の関連性や重要性のために、参照や記述の目的で目立たせたり強調したりする文字列を表します。

マークアップ例

This is a<mark>pen</mark>.

実行結果

This is a pen.

the em element

説明

強調されたテキストを示します。この要素は入れ子にすることができ、入れ子の段階に応じてより強い程度の強調を表すことができます。

マークアップ例

This is a<em>pen</em>.

実行結果

This is a pen.
